## Exemple : checkout pack

```bash
curl -X POST https://api.kiriku.app/v1/billing/checkout \
  -H "Authorization: Bearer <JWT>" \
  -H "Content-Type: application/json" \
  -d '{ "planId": "pack_growth" }'
```

## Exemple : abonnement

```bash
curl -X POST https://api.kiriku.app/v1/billing/subscribe \
  -H "Authorization: Bearer <JWT>" \
  -H "Content-Type: application/json" \
  -d '{ "plan": "growth", "phone": "+221771234567" }'
```
