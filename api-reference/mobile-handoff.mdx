## Endpoints

Voir la documentation détaillée de chaque endpoint :

- [Create session](/api-reference/endpoint/create-session) - `POST /v1/mobile-handoff/session`
- [Get session status](/api-reference/endpoint/get-session) - `GET /v1/mobile-handoff/session/:sessionId`
- [Upload file](/api-reference/endpoint/upload-file) - `POST /v1/mobile-handoff/upload/:sessionId`
- [Consume file](/api-reference/endpoint/consume-file) - `GET /v1/mobile-handoff/consume/:sessionId`

Tous ces endpoints sont publics et ne nécessitent pas d'authentification.
