Kiriku supports a growing list of document types across West Africa.

## Senegal (SN)

| Document Type | Slug | Description |
| :--- | :--- | :--- |
| **CNI (CEDEAO)** | `cni-senegal` | National Identity Card (front & back) |
| **Passport** | `passeport-senegal` | Biometric Passport |
| **Permis de Conduire** | `permis-conduire-cedeao` | Driving License (CEDEAO format) |
| **NINEA** | `ninea` | Tax Identification Number Certificate |
| **Extrait de Naissance** | `extrait-naissance` | Birth Certificate |
| **Carte de Séjour** | `carte-sejour` | Residence Permit |

## Generic / Auto

If you don't know the document type in advance, you can use the `auto` mode.

| Slug | Description |
| :--- | :--- |
| `auto` | Automatically detects the document type and extracts relevant fields. |
