---
title: "OCR providers"
description: "Liste des fournisseurs OCR disponibles"
---

Les fournisseurs dépendent de votre configuration serveur. Utilisez l’endpoint suivant pour voir les fournisseurs actifs :

```bash
curl https://api.kiriku.app/v1/ocr/providers
```

## Fournisseurs possibles

- `openrouter`
- `paddle-ocr`
- `cloudflare-ai`
