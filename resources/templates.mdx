Les templates définissent comment extraire les champs d’un document.

## Templates intégrés

Vous pouvez lister les templates disponibles :

```bash
curl https://api.kiriku.app/v1/templates \
  -H "X-API-Key: sk_live_..."
```

## Templates personnalisés (plan growth)

La création et la modification exigent un plan **growth** ou supérieur.

Exemple minimal :

```json
{
  "slug": "custom-invoice",
  "name": "Custom invoice",
  "country": "SN",
  "fields": [
    {
      "key": "invoice_number",
      "label": "Invoice number",
      "type": "string",
      "required": true
    }
  ]
}
```
